LabMec is a leading company in the construction of CNC machines, robots and automated solutions for the industrial and research sector. To enhance their online positioning, I created a cutting-edge website that combines high performance with a modern and functional design.

> If you want to see the live site, visit [labmec.it](https://www.labmec.it)

### Project Overview

Starting from a collaboration with an agency, the project aimed to develop a website that reflected LabMec's innovation and technological excellence. I developed the entire platform using Next.js and Tailwind CSS, integrating a bridge with the Notion API to leverage Notion as CMS. This solution enabled dynamic content and metadata management, significantly optimizing SEO.

### Technology Stack

The site was developed using [Next.js](https://nextjs.org/) to guarantee high performance and a scalable structure, while [Tailwind CSS](https://tailwindcss.com/) was used for a modern and customized design. Integration with the Notion API allows using Notion as CMS, offering a simple and dynamic content management system. Furthermore, the system leverages Next.js capabilities to dynamically generate metadata, optimizing SEO indexing and ensuring a high-level online presence.
